# EventCalendar

### Описание проекта
Проект представляет собой календарь событий с авторизацией. В рамках проекта реализована только Front-end часть приложения, использованы "моковые" аккаунты.

### Стэк
- TS (JS)
- React
- Redux
- Ant Design

### Инструкция к использованию
1. Скачать проект на свой ПК
2. Открыть терминал в папке с проектом
3. Ввести команды:
   - npm install
   - npm start
